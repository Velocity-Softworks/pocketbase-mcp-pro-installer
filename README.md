# PocketBase MCP Pro

AI-powered PocketBase management for Claude, Cursor and other MCP clients.

## Installation

```bash
npx pocketbase-mcp-pro
```

That's it. The installer will:
1. Ask for your license key
2. Download and install the latest version
3. Show you the config snippet to paste into your MCP client
